# No Raw Data Redistribution

This repository does not contain raw participant data, raw electromyography arrays, videos, original dataset archives, posterior-trace archives, or model checkpoints.

The benchmark uses public third-party datasets. Users must obtain those datasets from the original sources and comply with the original dataset licenses, access conditions, and ethics documentation.

The included outputs are aggregate tables, unit-level audit summaries, figure source summaries, and risk-boundary diagnostics needed to verify the paper-level results without redistributing raw data.
